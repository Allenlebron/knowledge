### Block 底层  
__Block(闭包，代码块)__: 带有 _局部变量_ 的 _匿名函数_。其本质是将要执行的代码进行包装，并且保存起来，等待需要执行的时候调用   

```
// OC 
int (^ blockName)() = ^(int num) {

}

// Swift  
```
